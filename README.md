# Proyecto lógica Katas de Python 🐍

Este repositorio contiene la resolución del proyecto final del módulo de Python del bootcamp de Data Analytics de The Power Education. 

El objetivo principal de este proyecto es demostrar el dominio de la lógica de programación mediante la resolución de un conjunto de 41 "Katas" (ejercicios prácticos de código), aplicando los fundamentos del lenguaje Python.

## 🛠️ Tecnologías y herramientas utilizadas
* **Lenguaje:** Python 3
* **Editor de código:** Visual Studio Code
* **Librerías / Módulos:** `functools` (para el uso de la función reduce) y `math`.

## 📋 Pasos seguidos durante el proyecto

Para abordar las 41 katas de forma ordenada y estructurada, he seguido la siguiente metodología de trabajo:

1. **Preparación del entorno:** Creación del archivo principal `Proyecto_katas.py` y estructuración del documento copiando todos los enunciados en forma de comentarios para mantener el orden.

2. **Agrupación por temáticas:**
   En lugar de hacerlos de forma estrictamente lineal, clasifiqué mentalmente los ejercicios para aplicar la misma lógica de una vez. Los bloques principales fueron:
   * **Tipos de datos y estructuras:** Manejo de listas, tuplas, diccionarios y strings.
   * **Programación Funcional:** Uso intensivo de las funciones integradas `map()`, `filter()`, `reduce()` y funciones anónimas `lambda`.
   * **Manejo de Excepciones:** Creación de bloques `try/except` y lanzamiento de errores personalizados con `raise` para asegurar que el programa no colapse ante entradas inesperadas de los usuarios (ej. dividir por cero o edades negativas).
   * **Programación Orientada a Objetos (POO):** Diseño de Clases (`Arbol`, `UsuarioBanco`) definiendo sus atributos en el método constructor `__init__` y creando métodos para alterar su estado.

3. **Desarrollo y Testing:**
   Resolución de cada kata individualmente, realizando pruebas por consola (`print`) para verificar que la salida era la esperada según las reglas lógicas del enunciado. Una vez validada, el código de prueba se eliminó para mantener la limpieza del archivo.

4. **Documentación del código:**
   Se han añadido comentarios explicativos (`#`) dentro de las funciones en los puntos más complejos o técnicos (como el uso de lambdas o el manejo de sets) para demostrar el entendimiento de la lógica aplicada, tal y como se requiere en las instrucciones.

## 📁 Estructura del repositorio

* `Proyecto_katas.py`: Archivo principal que contiene la resolución de todos los ejercicios. Cada kata está separada e identificada con su enunciado comentado.
* `README.md`: Este archivo, que documenta el proceso y los pasos seguidos.

## 🎓 Requisitos demostrados en este proyecto
A lo largo del código se puede comprobar la aplicación práctica de:
- Manejo de tipos de datos básicos y funciones incorporadas.
- Manejo de estructuras de datos en Python y sus métodos.
- Manejo de condicionales (`if`, `elif`, `else`).
- Manejo de estructuras de iteración (`for`, `while`).
- Manejo de funciones en Python (`def`, `lambda`).
- Manejo de Clases y entendimiento de la programación orientada a objetos (POO).
- Uso de módulos (`math`, `functools`).
- Buenas prácticas (código omentado, nombres de variables descriptivos, indentación correcta).
